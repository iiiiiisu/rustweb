# Rust Web 学习

### 背景
通过开发Web应用来学习Rust，从官网的教程开始，逐步填充自己的想法，并不断的完善。

### 用法
直接使用Git克隆本项目，使用cargo run即可运行。

> 请确认已正确安装Rust。
